SOC

Right now my soc i perfectly talored to my environment i have many servers that need protecting and using my soc i know they will be. The soc contains of a wazuh seim that sends logs to n8n which is an automation software this then takes action on the alerts from the seim in many ways firstly all alerts i find important (failed logins, added files, etc) are sent to me in a notification on discord. Then depending on the alert action gets taken the firstly if the alert is about a new file the file is scanned using virustotal. If it gets detected as malware the n8n server will sign into the machine using ssh and delete the file for me. Then if there is many login attempts the n8n server will block traffic on port 22. and there are many more actions the soc can take

the soc also contains an opencti server for documenting cases of attacks or malwar in my servers and there is also a velociraptor for forensics

the soc is a work in progress and i am constantly adding new features

as of the time i'm writing this i am experimenting with cuckoo sandbox for malware analysis which right now i still do manually on some vm's i have specifically made for that also in the soc

I have also recently added some ai enrichment to my reports on opencti and my discord alerts where i can right press an alert and press "send to n8n" and the message will be sent back and an ai agent from my local ai server will be used to enrich the alert in n8n and it will be sent back

what i learned:

  • SIEM rule tuning and alert triage (Wazuh)
  • Automated response and orchestration with n8n, including safe SSH operations
  • Malware triage and analysis using VirusTotal and Cuckoo Sandbox
  • Threat intelligence integration and case management with OpenCTI
  • Endpoint forensics and live response using Velociraptor
  • Incident response playbooks, alert prioritization, and false-positive reduction
  • Scripting and automation (Python/Bash) for repeatable containment workflows
  • Logging architecture, secure network hardening, and continuous monitoring
